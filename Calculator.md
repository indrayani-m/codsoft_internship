# Simple Calculator

## Description
A console-based calculator that performs basic arithmetic operations such as addition, subtraction, multiplication, and division using C++ classes and methods.

## Features
- Four arithmetic operations
- Menu-driven interface
- Object-oriented structure

## Concepts Used
- Classes and methods
- Switch-case
- Loops

## How to Run
1. Open [GDB Online Compiler](https://www.onlinegdb.com/).
2. Copy and paste the `Simple_Calculator.cpp` code.
3. Click **Run** to execute the program.
