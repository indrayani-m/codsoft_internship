# Number Guessing Game

## Description
A console-based game where the player guesses a randomly generated number within a user-defined range. The program gives hints until the correct number is guessed.

## Features
- User-defined range
- Random number generation
- Feedback for each guess

## Concepts Used
- Loops
- Conditional statements
- Random number generation

## How to Run
1. Open [GDB Online Compiler](https://www.onlinegdb.com/).
2. Copy and paste the `Number_Guessing_Game.cpp` code.
3. Click **Run** to execute the program.
