# Codesoft Internship Projects
This repository contains four C++ mini projects created during my Codesoft Internship. Each project demonstrates core programming concepts and object-oriented design using C++.

## Projects Included
1. Number Guessing Game  
2. Simple Calculator  
3. Library Management System  
4. To-Do List App

## Tech Stack
- Language: C++
- Compiler: GDB Online Compiler
- Concepts Used: OOP, loops, conditionals, vectors, time library

## How to Run
1. Open [GDB Online Compiler](https://www.onlinegdb.com/).
2. Copy the code from the required `.cpp` file.
3. Paste it into the editor.
4. Choose **C++** as the language.
5. Click **Run** to execute the program.
