# To-Do List App

## Description
A console-based task management application that allows users to add, view, mark, and delete tasks.

## Features
- Add and view tasks
- Mark tasks as done
- Remove completed tasks

## Concepts Used
- Classes and objects
- Vectors
- Loops and conditionals

## How to Run
1. Open [GDB Online Compiler](https://www.onlinegdb.com/).
2. Copy and paste the `To_Do_List_App.cpp` code.
3. Click **Run** to execute the program.
