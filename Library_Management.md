# Library Management System

## Description
A simple console-based program to manage library operations such as adding, borrowing, returning, and searching for books. It also calculates fines for late returns.

## Features
- Add, borrow, return, and search books
- Fine calculation for late returns
- Display of available books

## Concepts Used
- Classes and objects
- Vectors
- Time library for fine calculation

## How to Run
1. Open [GDB Online Compiler](https://www.onlinegdb.com/).
2. Copy and paste the `Library_Management_System.cpp` code.
3. Click **Run** to execute the program.
